# download-music

## 功能

1. 搜索歌曲并获取信息
    - 歌曲名
    - 专辑
    - 专辑排序
    - 创作者
    - 发行时间
    - 时长
    - 比特率
2. 下载歌曲

## 支持

- [ ] 网易云
- [x] 酷狗
- [ ] 酷我
- [ ] QQ
